# Netris by Alex

## Goals

* head to head IP networking
* different gameplay modalities - deathmatch or rounds
* * I clear a row, you get a row
* * Reach the goal score first to finish a round (rounds require store)
* * Round winner gets more points ()
* between rounds store
* store powerups require fully extensible plugin architecture
* powerup ideas:
* * swap next pieces with opponent
* * reverse/rotate opponent's input bindings
* * flip/rotate opponent's display
* * cause opponent's piece to rotate per drop (needing correction)
* * give yourself an I piece
* * let users make their own plugin (mods)
